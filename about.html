ABSTRACT

This project aims to design an interactive 3D simulation using virtual reality technology.
to provide an opportunity for anyone who wishes to visit the Aqaba Aquarium to do so without the need to be physically present.
With a focus on accuracy in design, the virtual museum is modeled to match the original museum. However, it intentionally does not provide 
the full experience,in order to encourage users to visit the physical museum and enjoy the real-life atmosphere. 

Firstly, the virtual museum provides an opportunity for anyone who cannot visit in person to explore it virtually, 
without the hardship of travel.
Secondly, the virtual museum can be used in education by offering virtual tours to school children. 
This eliminates potential risks and logistical issues since schools might not have the means to organize such trips. 
Furthermore, the project relies on a set of tools, which are:
tool use
Blender designing 3D scenes
unity simulations and interactions
visual studio programming
